HATEOAS

Single-entity endpoints carry a _links envelope you can follow without constructing URLs manually:

{
  "id": "1101",
  "province_id": "11",
  "name": "KABUPATEN SIMEULUE",
  "_links": {
    "self":     "/api/regency/1101.json",
    "parent":   "/api/province/11.json",
    "children": "/api/districts/1101.json"
  }
}

List endpoints omit _links per row to keep array responses compact.
